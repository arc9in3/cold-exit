# Art Requests — Hideout Cast + Backgrounds

Cast direction: **mostly dark / sultry / weathered female**; a few males for variety. Cold Exit's existing palette is muted, desaturated, gritty realism (Disco Elysium / Beksinski / cold cyberpunk noir). Each character has a portrait **and** a paired room background for the hideout UI behind them.

Standing rules baked into every prompt below:

- single centered subject, head-and-shoulders portrait or 3/4 view
- **no duplicates, no rotation grids, no contact sheets, no character sheets**
- 1024×1024, solid or atmospheric single background (NOT a multi-panel collage)
- desaturated palette, dim warm key + cool fill, harsh single light source
- film-grain, painted texture, no clean digital sheen

## How to use

For each character above, post **two** Discord messages in `#art-requests-2d`:
1. The **portrait prompt** under a heading like `## 3. THE GUNSMITH — portrait`
2. The **room prompt** under `## 3. THE GUNSMITH — room`

Tag with `[character]` and `[background]` in the title so the renames stay tidy. Paste **only the prompt blockquote and the negative line** into the request — Artie doesn't need the cast notes, those are for our reference.

Once a result lands and you like it, drop it into `Assets/generated/` with the naming convention `npc-<role>.png` for portraits and `room-<role>.png` for backgrounds. The hideout UI / shop UI can then reference them by deterministic path.
